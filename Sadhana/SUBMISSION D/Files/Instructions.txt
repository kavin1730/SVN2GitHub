****** Multi-Channel ADC ******

****** INSTRUCTIONS ******

1) There are three tabs present. The Home Page is the working side. The ADC's available shows the types that are defined in the file.

2) On running the VI, the ADC type can be chosen.

3) The details of the ADC chosen will be displayed.

4) On pressing START the data acquisition will be started and the data will be plotted.

5) The file path when chosen valid and pressed OK then the digital data will be logged in the text file.

6) If Edit Configuration button is pressed the configuration of the current chosen ADC can be edited.

7) On pressing Confirm the changes are applied.

8) The Channel selectors are present so that particular channels can be viewed.

9) Panel close will stop the VI from running.
